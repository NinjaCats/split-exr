### How it works

[![Split exr layers]()](https://vimeo.com/248760837 "Split exr layers")

### Installation

`git clone https://github.com/artemhlezin/split-exr.git`

add `split-exr` folder to your nuke plugin path

init.py
```
import nuke

nuke.pluginAddPath("./path/to/folder/split-exr")
```
